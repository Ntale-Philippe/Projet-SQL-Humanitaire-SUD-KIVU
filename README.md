# Analyse des Distributions Humanitaires (SQL) - Sud-Kivu

## Description du Projet
Ce projet simule une base de données de suivi humanitaire pour la province du Sud-Kivu (RDC). 
L'objectif est d'extraire des indicateurs clés sur les distributions effectuées dans les sites de déplacés.

## Compétences SQL démontrées :
- Création de base de données relationnelle.
- Jointures multiples (Triple Joins).
- Agrégations complexes (SUM, AVG, COUNT).
- Filtrage avancé (HAVING, WHERE).

## Structure des données
Le projet utilise trois tables principales :
1. **Projets** : Détails financiers et sectoriels.
2. **Sites** : Localisation géographique (Territoires).
3. **Distributions** : Données opérationnelles et quantités.
